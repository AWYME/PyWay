Платформа для изучения Python. Веб-приложение на Flask позволяет проходить уроки и решать практические задачи с автоматической проверкой кода прямо в браузере.

Возможности:
    Интерактивные уроки по Python
    Встроенный редактор кода с проверкой
    Система отслеживания прогресса
    Панель управления для преподавателей
    
Структура проекта:
    main.py — основное приложение Flask
    admin.py — административные функции
    static/ — CSS, JavaScript, изображения
    templates/ — HTML-шаблоны
    requirements.txt — зависимости Python
    
Использование:
    Для студентов: регистрация, выбор урока, изучение теории, решение задач
    Для преподавателей: вход в админ-панель, добавление уроков и упражнений
    
Технологии:
    Backend: Python, Flask
    База данных: SQLite
    Frontend: HTML, CSS, JavaScript
